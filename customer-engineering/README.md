# Customer Engineering Skills

Skills for Customer Engineering, Customer Success, and post-sales teams: account intelligence, meeting prep, health and risk workflows, renewals, and growth.

Each skill is a single self-contained `SKILL.md`. Bodies reference tools by capability name so they run on any Agent Skills-compatible agent; `metadata.quivly-tools` carries exact Quivly tool-catalog IDs so imports into Quivly attach tools automatically. In Claude Code or claude.ai, connect the Quivly MCP connector and the same skills work as-is.

## Skills by tier

**Daily drivers**
- `meeting-prep` — pre-call brief for any upcoming customer call
- `account-360` — full account snapshot on demand
- `post-call-followup` — turn the latest call into actions and a recap
- `ticket-pulse` — support health read and escalation candidates
- `inbox-triage` — what needs my attention today across my book

**Weekly rhythms**
- `portfolio-digest` — book-of-business movers, droppers, upcoming renewals
- `usage-drop-investigation` — why usage fell and what correlates
- `health-drop-diagnosis` — decompose a health score drop
- `at-risk-report` — leadership roll-up of at-risk accounts
- `renewal-forecast` — renewals due next quarter with confidence reads

**Lifecycle-triggered**
- `sales-handoff` — deal-close → CSM handoff brief
- `kickoff-prep` — onboarding kickoff agenda and stakeholder map
- `onboarding-stall` — time-to-value check for new accounts
- `renewal-risk` — 60-day-out renewal read
- `churn-save-plan` — red account deep dive → save plan
- `champion-change` — sponsor left: gap analysis and re-entry plan
- `escalation-brief` — executive escalation one-pager
- `customer-health-review` — full structured health review

**Quarterly / growth**
- `qbr-prep` — value story, usage narrative, renewal posture
- `expansion-scan` — upsell and cross-sell signals
- `advocacy-finder` — reference and case-study candidates
